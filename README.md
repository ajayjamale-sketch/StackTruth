# StackTruth — Production-Ready Frontend MVP

StackTruth is a premium "Cyber Intelligence" themed developer knowledge and collaboration platform. Built with React 18, Vite, TypeScript, and TailwindCSS, this MVP delivers a high-fidelity experience for developers, experts, recruiters, and administrators.

## 🚀 Key Features

- **Role-Based Workspaces**: Specialized dashboards for Developers, Experts, Recruiters, and Admins.
- **Verification System**: Verified profiles and expert auditing protocols.
- **Global Discussion Hub**: Community-driven Q&A with bounty support.
- **AI-Assisted Workflows**: Integrated AI assistant for code analysis and problem-solving.
- **Hiring Portal**: Advanced recruitment pipeline and talent discovery.
- **Enterprise Design System**: Custom glassmorphism, grid-patterns, and ultra-responsive layout.

## 🛠 Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: TailwindCSS (Custom configuration)
- **Icons**: Lucide React
- **Analytics**: Recharts
- **Language**: TypeScript

## 🌗 Theme Support

The platform features a robust light/dark mode system based on the `Slate` and `Emerald` color palettes, ensuring maximum readability and a professional aesthetic across all devices.

## 📁 Project Structure

- `src/components`: Reusable UI elements, layout components, and feature-specific modules.
- `src/pages`: 20+ functional pages including dashboards, auth, and legal.
- `src/contexts`: Global state management for Authentication, Theming, and Notifications.
- `src/constants`: Centralized mock data and configuration.
- `src/types`: TypeScript definitions for project-wide type safety.

## 🏗 Setup & Installation

1. Clone the repository
2. Run `npm install`
3. Start development server: `npm run dev`
4. Build for production: `npm run build`
